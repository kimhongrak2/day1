이번 주 운영 메모

- 월요일 오전 통관 문의가 몰리면서 물류운영팀 응답이 지연됨.
- 화요일 수출서류 처리 중 거래처 코드 불일치가 2건 발생함. 다음 주에는 거래처 코드 확인 절차를 먼저 거칠 필요가 있음.
- 수요일 계약검토 2건이 법무 확인 대기 상태로 넘어감. 금요일 오전까지는 회신이 오지 않음.
- 목요일 입고 예정일 변경 문의가 늘어 재고관리팀 확인 시간이 길어짐.
- 목요일 통관대응 중 HS 코드 재확인이 필요한 건이 있었음. 같은 유형이 반복될 가능성이 있음.
- 금요일 긴급 견적 요청 3건이 들어왔고, 그중 1건은 다음 주 월요일 오전까지 회신해야 함.
- 금요일 재고 수량 불일치가 2건 확인됨. 원인은 입고 처리 시점 차이로 추정됨.
- 단순 참고: 다음 주 화요일 팀 회의실 예약 완료.
- 단순 참고: 사내 교육 신청 마감 공지 전달 완료.
